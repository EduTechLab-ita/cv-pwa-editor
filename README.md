## ✨ Caratteristiche Principali

### 📱 **Progressive Web App**
- **Installabile** come app nativa su qualsiasi dispositivo
- **Funziona offline** - crea il tuo CV anche senza internet
- **Aggiornamenti automatici** quando disponibili

### 💾 **Gestione Dati Sicura**
- **Salvataggio automatico** - non perdi mai il tuo lavoro
- **Privacy totale** - i dati restano solo sul tuo dispositivo
- **Export/Import** completo per backup e condivisione
- **Reset template** per iniziare da capo quando vuoi

### 🎨 **Design Professionale**
- **Layout Europass** standard per CV professionali
- **Responsive design** - perfetto su desktop, tablet e mobile
- **Anteprima real-time** - vedi subito come sarà il risultato
- **Stampa PDF** ottimizzata per colloqui di lavoro

### 🔧 **Funzionalità Avanzate**
- **7 sezioni complete**: Info personali, Profilo, Esperienza, Formazione, Competenze, Formatore, Impostazioni
- **Validazione automatica** dei campi
- **Gestione dinamica** di esperienze e formazioni
- **Backup JSON** per trasferire i dati tra dispositivi

## 🛠️ Come Usare

### 💻 **Utilizzo Online (Consigliato)**
1. **Vai su** [edutechlab-ita.github.io/cv-pwa-editor](https://edutechlab-ita.github.io/cv-pwa-editor/)
2. **Compila** le sezioni del tuo CV
3. **Esporta** in PDF o salva i dati in JSON
4. **Installa** l'app per uso offline (opzionale)

### 📱 **Installazione PWA**
- **Desktop**: Click sull'icona "Installa" nella barra indirizzi del browser
- **Mobile Android**: Banner automatico "Aggiungi alla schermata Home"
- **Mobile iOS**: Safari → Condividi → "Aggiungi a schermata Home"

### 💾 **Backup e Condivisione**
- **Export**: Impostazioni → Export → Backup JSON
- **Import**: Impostazioni → Import Dati → Carica file JSON
- **PDF**: Bottone "Stampa PDF" per download immediato
